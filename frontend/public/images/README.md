# Imagenes del catalogo

Aqui van las imagenes finales del catalogo que se sirven desde el frontend.

Las rutas deben coincidir exactamente con el valor de `image_url` guardado en Supabase.

Ejemplos:

- `/images/dolls/burnice-white.jpg`
- `/images/accessories/nitro-fuel.jpg`
- `/images/keychains/mini-ellen-joe.jpg`
- `/images/combos/combo-mini-icons.jpg`

Si cambias el nombre de un archivo o lo mueves de carpeta, actualiza tambien el campo `image_url` correspondiente en Supabase.
